# Maven Network
A simple GUI aplication for creating mnist prediction network
Implements Deep neural network with different activation functions
Has drawing canvas for MNIST predictions(Warning! some numbers maybe harder to predict)

# Features 
- Neural Network
- drawing canvas
- testing data
- Softmaxcrossentropy 

# Versions
- java 25

# Instalation 
- cloning this repository
- adding csv files to resource folder from this site https://www.kaggle.com/datasets/oddrationale/mnist-in-csv
- Build with maven
